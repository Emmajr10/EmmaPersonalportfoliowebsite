Project Features
Responsive Portfolio Website

The portfolio was designed with a clean and simple layout to ensure easy navigation and accessibility. 
The site is structured into clearly defined sections that present information about my background and work.

The main sections of the website include:
Home
 It Provides a brief introduction about who I am and highlights my professional focus in IT support and technology.

About

This section includes details about my educational background, certifications, and technical skills.

Projects

Displays selected technical projects that demonstrate my hands-on experience with networking, programming, and system administration. Each project includes a description and a link to its corresponding repository on GitHub.

Resume

Visitors can download my most recent resume directly from the website. 
The resume highlights my certifications, experience, and technical competencies.

Contact

The contact section contains a form allowing visitors to reach out to me for professional opportunities or collaboration.
